# 🏥 Doctor Appointment Management App

This is a Python-based app to manage doctor appointments with full CRUD operations.  
Users can create, view, update, and delete appointments.  
It helps clinics or users keep track of doctor visit schedules easily.

## 🔧 Features
- 🆕 Create a new appointment  
- 📋 View all appointments  
- ✏️ Update existing appointment  
- ❌ Delete an appointment
